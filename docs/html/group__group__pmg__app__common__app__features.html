◆ CY_APP_USB_ENABLE

#define CY_APP_USB_ENABLE   (0u)

Enable/disable the USB feature.

This macro should be set to '1' for solutions that need the USB (Vendor, Billboard and HID) feature support. If this option is disabled, then CCG_BB_ENABLE, CY_APP_USB_HID_INTF_ENABLE and any other macro for enabling USB interfaces should be disabled. CCG_BB_ENABLE should not be disabled if the alternate mode support is needed in the Dock solution.

◆ CY_APP_WATCHDOG_RESET_PERIOD_MS

#define CY_APP_WATCHDOG_RESET_PERIOD_MS   (750u)

Watchdog reset period in milliseconds.

A periodic timer (CY_PDUTILS_WATCHDOG_TIMER) is run using this as a period. This timer resets the device if the main loop has not been run as expected for three consecutive times.
