◆ Cy_App_SystemSleep()

bool Cy_App_SystemSleep ( cy_stc_pdstack_context_t *  ptrPdStackContext,
cy_stc_pdstack_context_t *  ptrPdStack1Context 
)

Function to place PMG device in power saving mode if possible.

This function places the PMG device in power saving Deep Sleep mode if possible. The function checks for each interface (PD, HPI, etc.) being idle and then enters into Sleep mode with the appropriate wakeup triggers. If the device enters Sleep mode, the function will only return after the device has woken up.

Parameters
ptrPdStackContextPointer to the PDStack context.
ptrPdStack1ContextPointer to the PDStack context of port 1.
Returns
True if the device went into Sleep; false otherwise.
