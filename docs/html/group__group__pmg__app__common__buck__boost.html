Buck-boost

General Description

The buck-boost driver provides APIs and data structures to control the external buck-boost controller.

This library supports two types of buck-boost controllers, namely MP4247 from MPS® and RT6190 from Richtek®. Only one controller driver can be enabled at a time. To enable the desired controller driver, its respective macro, either CY_APP_BUCKBOOST_MP4247_ENABLE or CY_APP_BUCKBOOST_RT6190_ENABLE should be enabled.

Features:

  • Supports controller initialization
  • Enable/disable output voltage
  • Set output current limit

configuration considerations

Buck-boost driver is built over the I2C master driver. I2C SCB block should be initialized as a master mode before using this driver. Refer to the CAT2 Peripheral Driver Library API reference manual for configuring the SCB block.

The following steps describe the method for enabling buck-boost driver in an application.

  1. Include buck_boost.h to get access to all functions and other declarations.
    #include "cy_app_buck_boost.h"
  2. Define the I2C SCB context data structure.
    cy_stc_scb_i2c_context_t i2cContext;
  3. Define the following data structures required by the buck-boost driver.
    #if CY_APP_BUCKBOOST_MP4247_ENABLE
    cy_stc_app_mp4247_context_t gl_mp4247ContextPort1 =
    {
    .i2cAddr = 0x67u,
    .scbBase = SCB4,
    .i2cContext = &i2cContext,
    .enableGpioPort = GPIO_PRT5,
    .enableGpioPin = 5U,
    .fbRatio = 15u
    };
    #elif CY_APP_BUCKBOOST_RT6190_ENABLE
    cy_stc_app_rt6190_context_t gl_rt6190ContextPort1 =
    {
    .i2cAddr = 0x2Cu,
    .scbBase = SCB4,
    .i2cContext = &i2cContext,
    .enableGpioPort = GPIO_PRT5,
    .enableGpioPin = 5U,
    };
    #elif CY_APP_BUCKBOOST_NCP81239_ENABLE
    cy_stc_app_ncp81239_context_t gl_ncp81239ContextPort1 =
    {
    .i2cAddr = 0x74,
    .scbBase = SCB4,
    .i2cContext = &i2cContext,
    .enableGpioPort = GPIO_PRT1,
    .enableGpioPin = 3U
    };
    #endif /* CY_APP_BUCKBOOST_MP4247_ENABLE */
  4. Initialize the buck-boost driver.
    /* Initialize buck boost controller. */
    #if CY_APP_BUCKBOOST_MP4247_ENABLE
    Cy_App_BuckBoost_InitPort1(&gl_mp4247ContextPort1);
    #elif CY_APP_BUCKBOOST_RT6190_ENABLE
    Cy_App_BuckBoost_InitPort1(&gl_rt6190ContextPort1);
    #elif CY_APP_BUCKBOOST_NCP81239_ENABLE
    Cy_App_BuckBoost_InitPort1(&gl_ncp81239ContextPort1);
    #endif /* CY_APP_BUCKBOOST_MP4247_ENABLE */
  5. Invoke the set voltage API to set the required voltage.
    /* Set port1 output voltage */
    Cy_App_BuckBoost_SetVoltPort1(5000);
