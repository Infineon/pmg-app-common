◆ Cy_App_Dmc_InitMetadata()

void Cy_App_Dmc_InitMetadata ( cy_stc_dmc_params_t *  params)

Initialize dock metadata.

When dock metadata is empty in DMC flash, form the dock metadata by querying various devices. When dock metadata is valid, copy the dock metadata contents from flash to RAM copy. Update the dock metadata in RAM from CDTT parameters and write back RAM copy into flash.

Parameters
paramsPointer to DMC parameters structure
Returns
None
