API Reference
HPI PD Response handler

General Description

This module provides APIs and data structures for read/write the PD response register.

These APIs are registered as application callbacks and used by the HPI middleware library.

Features:

  • Read/write battery status
  • Read/write battery capabilities
  • Read/write source Info and PD revision message

Configuration considerations

The following steps describe the usage of these APIs.

  1. Include cy_app_hpi.h to get access to all functions and other declarations.
    #include "cy_pdstack_common.h"
    #include "cy_app_hpi.h"
    #include "cy_hpi.h"
  2. Make sure the Host Processor Interface (HPI) middleware library is configured and initialized before using these APIs.
    See the HPI middleware API reference manual
  3. Initializes the HPI application callback struct member hpi_rw_pd_resp_data.
    cy_stc_hpi_app_cbk_t hpiAppCbk =
    {
    #if (CY_HPI_RW_PD_RESP_MSG_DATA)
    .hpi_rw_pd_resp_data = Cy_App_Hpi_HandlePdRespDataRw,
    #endif /* (CY_HPI_RW_PD_RESP_MSG_DATA) */
    };
  4. The following example demonstrates the steps required to store data into the response buffer.
    /* Get the USBPD Revision message. */
    cy_stc_pdstack_dpm_ext_status_t* ptrDpmExtStat = &(glPdStackPort0Ctx.dpmExtStat);
    uint32_t pdRevison = ptrDpmExtStat->revision.val;
    /* Write the USBPD Revision message to PD Response message data buffer */
    uint8_t revision_msg[8] = {CY_APP_PD_RESP_ID_PD_REV_MSG,
    0x00,
    CY_APP_PD_RESP_CMD_REV_MSG_LEN,
    0x00,
    (uint8_t) (pdRevison),
    (uint8_t) (pdRevison>>8u),
    (uint8_t) (pdRevison>>16u),
    (uint8_t) (pdRevison>>24u)};
    Cy_App_Hpi_HandlePdRespDataRw(&glHpiContext, &glPdStackPort0Ctx, revision_msg);
  5. The following example demonstrates the steps required to retrieve data from the response buffer.
    cy_stc_app_pd_resp_stored_t *ptrPdResp;
    uint32_t pdRevision = 0x00u;
    /* Revision message stored in the memory is 4 bytes in size.*/
    cy_stc_app_pd_resp_data_t pdRevisionResp;
    pdRevisionResp.respId = CY_APP_PD_RESP_ID_PD_REV_MSG;
    pdRevisionResp.respLen = CY_APP_PD_RESP_CMD_REV_MSG_LEN;
    /* Retrieve the source info response with 4 bytes as length.*/
    ptrPdResp = Cy_App_Hpi_RetrievePdRespData(&pdRevisionResp, 0x00u);
    if(NULL != ptrPdResp)
    {
    /* Get the USBPD revision from stored response. */
    memcpy(&pdRevision, (uint8_t *)&ptrPdResp->respData, 0x04);
    }
