I2C master

General Description

The I2C master provides APIs for transferring data to/from I2C slave device registers.

This module is built on top of the I2C SCB driver that is available as part of PDL CAT2.

Features:

  • Read and Write Registers

Configuration considerations

The following steps describe the way to enable the I2C master.

  1. Include cy_app_i2c_master.h to get access to all functions and other declarations.
    #include "cy_pdl.h"
    #include "cy_app_i2c_master.h"
  2. Configure the SCB block as I2C master using Device Configurator link available in the ModusToolbox™ quick panel. Refer to the PDL API reference.
  3. Define the I2C SCB context data structure
    cy_stc_scb_i2c_context_t gl_I2cMasterCtx;
  4. Invoke the read API to read data from the slave device registers
    uint16_t read_reg_addr = 0x2000;
    uint8_t read_buff[4];
    /* Read from the slave device register. */
    Cy_App_I2CMaster_RegRead(SCB0,
    0x08,
    (uint8_t *)&read_reg_addr,
    2U,
    read_buff,
    4U,
    &gl_I2cMasterCtx);
  5. Invoke the write API to write data to the slave device registers
    uint16_t write_reg_addr = 0x2000;
    uint8_t write_buff[4] = {0x01, 0x02, 0x03, 0x04};
    /* Write to the slave device register. */
    Cy_App_I2CMaster_RegRead(SCB0,
    0x08,
    (uint8_t *)&write_reg_addr,
    2U,
    write_buff,
    4U,
    &gl_I2cMasterCtx);
