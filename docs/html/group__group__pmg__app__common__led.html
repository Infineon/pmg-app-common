LED controller

General Description

The LED controller provides APIs and data structures to control a user LED connected to a GPIO.

This module uses a software timer to control the LED in various modes.

Features:

  • ON/OFF mode
  • Blink mode
  • Breathing mode

configuration considerations

The following steps describe the method for enabling the LED controller in an application.

  1. Include cy_app_led_ctrl.h to get access to all functions and other declarations
    #include "cy_pdl.h"
    #include "cy_pdutils.h"
    #include "cy_app_led_ctrl.h"
  2. Configure the GPIO as output pin using Device Configurator link available in the ModusToolBox™ quick panel.
  3. Define the LED controller context data structure
    #if CY_APP_LED_CONTROL_ENABLE
    cy_stc_led_ctrl_context_t gl_LedCtrlCtx;
    #endif /* CY_APP_LED_CONTROL_ENABLE */
  4. Initialize the context structure members
    gl_LedCtrlCtx.port = GPIO_PRT3;
    gl_LedCtrlCtx.pinNum = 3U;
    gl_LedCtrlCtx.timerId = APP_LED_CTRL_TIMER;
    gl_LedCtrlCtx.ledOrientation = false;
    gl_LedCtrlCtx.blink_rate = 1000U;
  5. Invoke the following API from main to initialize the LED controller
    /* Initialize LED controller */
    Cy_App_LedCtrl_Init(&gl_LedCtrlCtx);
  6. Use the following API to change the LED control mode
    /* Switch LED mode to ON state. */
    Cy_App_LedCtrl_SwitchMode((cy_en_led_ctrl_mode_t)CY_APP_LED_CTRL_LED_ON, &gl_LedCtrlCtx);
