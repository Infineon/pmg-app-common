cy_en_app_status_t Cy_App_LedCtrl_Init ( cy_stc_led_ctrl_context_t *  ledCtrlCtx)

This function is used to initialize the LED control module.

During initialization of the module, the LED connected to the GPIO will be turned OFF. The LED control context should be initialized before calling this function.

Parameters
ledCtrlCtxPointer to the LED control context
Returns
CY_APP_STAT_BAD_PARAM if a bad parameter is passed as argument CY_APP_STAT_SUCCESS otherwise

This function is used to change the mode of LED.

Parameters
ledModeThe desired mode of operation
ledCtrlCtxPointer to the LED control context
Returns
CY_APP_STAT_NO_STATE_CHANGE if the current mode of LED operation is same as the new mode requested CY_APP_STAT_BAD_PARAM if a bad parameter is passed as argument CY_APP_STAT_SUCCESS if the mode change was successful
