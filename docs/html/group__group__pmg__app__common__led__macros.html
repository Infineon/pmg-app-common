◆ CY_APP_LED_CTRL_BREATHING_COUNT

#define CY_APP_LED_CTRL_BREATHING_COUNT   (10u)

The duty cycle of PWM generated on the GPIO is changed in steps of 1 ms.

This macro defines the maximum number of steps used to change the duty cycle from 0 to 100% and vice versa.
