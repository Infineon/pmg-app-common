cy_en_debug_status_t Cy_App_Debug_Init ( CySCB_Type *  base,
cy_stc_scb_uart_config_t const *  config,
uint32_t  flash_addr,
uint32_t  backup_flash_addr,
uint16_t  timerID,
uint16_t  deferTime,
cy_stc_pdutils_sw_timer_t *  timerCtx 
)

This function is used for the initialization of the debug module.

Internally, it calls functions to initialize UART and flash logging.

Parameters
baseThe pointer to the UART SCB instance
configThe pointer to SCB UART configuration structure
flash_addrThis parameter specifies the address of internal flash that is used for flash logging.
backup_flash_addrThis parameter specifies the address of internal flash that is used for storing backup of flash logs.
timerIDTimer ID used for flash logging
deferTimeDefer time for flash writes
timerCtxPointer to timer context used for flash logging
Returns
CY_APP_DEBUG_STAT_BAD_PARAM if a bad parameter is passed. CY_APP_DEBUG_STAT_UART_INIT_FAILED if UART initialization failed. CY_APP_DEBUG_STAT_SUCCESS if the operation is successful.

This function will internally call UART and flash logging functions for logging the failures.

Parameters

is_stdThis parameter is used to specify the is_std field in the header of UART structure
Returns
CY_APP_DEBUG_STAT_SUCCESS if the operation is successful CY_APP_DEBUG_STAT_NOT_READY if flash log module or UART module is not initialized CY_APP_DEBUG_STAT_BAD_PARAM if an invalid evt is used CY_APP_DEBUG_STAT_FLASH_UPDATE_FAILED if flash write failed

◆ Cy_App_Debug_FlashLog()

cy_en_debug_status_t Cy_App_Debug_FlashLog ( uint8_t  port,
cy_en_debug_opcodes_t  evt,
uint8_t  failure_info 
)

This function handles flash log events.

It determines the offsets within the flash where the logs are to be updated and modifies the RAM copy of flash logs. The modified RAM copy of flash logs will be used to update the flash logs.

Parameters
portThis port specifies the port on which the error occurred
evtThe opcode corresponding to the error cy_en_debug_opcodes_t
failure_infoThe 1 byte data which is logged along with the opcode in internal flash
Returns
CY_APP_DEBUG_STAT_NOT_READY if flash log module is not initialized CY_APP_DEBUG_STAT_FLASH_UPDATE_FAILED if flash write failed CY_APP_DEBUG_STAT_SUCCESS if the operation is successful
