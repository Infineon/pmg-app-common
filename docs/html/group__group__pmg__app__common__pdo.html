PDO handling

General Description

The PDO handler provides APIs to evaluate the source capabilities advertised by port partner and identify the optimal power contract to be entered into and evaluate a PD request data object and determine whether to accept or reject the request.

These APIs are registered as application callbacks and used by the PDStack middleware library.

Configuration considerations

The following steps describe the usage of these APIs:

  1. Include cy_app_pdo.h to get access to all functions and other declarations.
    #include "cy_app_pdo.h"

  2. Initialize the functions to the application callback structure.
    /* Register Application callback functions with the PdStack Middleware.
    */
    static const cy_stc_pdstack_app_cbk_t app_callback =
    {
    .eval_src_cap = Cy_App_Pdo_EvalSrcCap,
    .eval_rdo = Cy_App_Pdo_EvalRdo,
    };

  3. Register the application callback to the PDStack middleware library. Refer to the Quick start guide section.
