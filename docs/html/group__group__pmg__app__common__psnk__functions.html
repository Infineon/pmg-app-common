◆ Cy_App_Sink_Disable()

void Cy_App_Sink_Disable ( cy_stc_pdstack_context_t *  context,
cy_pdstack_sink_discharge_off_cbk_t  snk_discharge_off_handler 
)

Disables the VBUS power sink path and discharge VBUS supply down to a safe level.

This function is called by the PDstack at times when the system is not allowed to draw power from the VBUS supply. The application can use this call to initiate a VBUS discharge operation so that a subsequent Type-C connection is speeded up. The snk_discharge_off_handler callback function is called once VBUS discharges down to vSafe0V.

Parameters
contextPointer to the PDStack context
snk_discharge_off_handlerSink Discharge fet off callback pointer
Returns
None
