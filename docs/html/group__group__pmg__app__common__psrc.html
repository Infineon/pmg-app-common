Provider path

General Description

Provider path handler provides APIs and data structures to control the power provider path and enables the fault detection.

These APIs are registered as application callbacks and used by the PDStack middleware library.

Features:

  • Set fault threshold voltage
  • Enable/Disable provider path
  • Set VBus voltage and current

Configuration considerations

The following steps describe the usage of these APIs:

  1. Include cy_app_source.h to get access to all functions and other declarations.
    #include "cy_pdl.h"
    #include "cy_app_source.h"

  2. Initialize the functions to the application callback structure.
    /* Register Application callback functions with the PdStack Middleware.
    */
    const cy_stc_pdstack_app_cbk_t app_callback =
    {
    #if (!CY_PD_SINK_ONLY)
    .psrc_set_voltage = Cy_App_Source_SetVoltage,
    .psrc_set_current = Cy_App_Source_SetCurrent,
    .psrc_enable = Cy_App_Source_Enable,
    .psrc_disable = Cy_App_Source_Disable,
    #endif /* (!CY_PD_SINK_ONLY) */
    };

  3. Register the application callback to the PDStack middleware library. See the Quick start guide section.
