PmgAppCommon Middleware Library
API Reference
System

General Description

The system modules allows user to perform system specific operation. The following functionality are provided by this.

  • System
    The system module provides APIs to set/get the firmware mode, version information, etc.
  • Bootloader
    The bootloader module provide APIs and data structures to validate and load the firmware image from flash.
  • Flash
    The flash APIs allows user to read and write the device flash memory.
  • Instrumentation
    The instrumentation module provides APIs to handle the device specific events (for example, watchdog timeout, hard fault).

configuration considerations

system

  1. Include cy_app_system.h to get access to all system specific functions and other declarations
    #include "cy_app_system.h"
  2. Invoke the APIs like the following to perform a system specific operations. Refer the system section for detailed information about usages and functionality of all other APIs.
    /* Variable for storing the firmware mode. */
    cy_en_sys_fw_mode_t fwMode;
    /* Set device firmware mode. */
    Cy_App_Sys_SetDeviceMode(CY_APP_SYS_FW_MODE_FWIMAGE_1);
    /* Get the device active firmware mode .*/
    fwMode = Cy_App_Sys_GetDeviceMode();

bootloader

  1. Include cy_app_boot.h to get access to all bootloader specific functions and other declarations
    #include "cy_app_boot.h"
  2. Define the firmware metadata data structure and firmware image status structure required by the bootloader APIs. Metadata table is placed at a fixed location for both firmware image-1 and image 2.
    /* Structure variable to store the firmware image status. */
    cy_stc_fw_img_status_t gl_imgStatus;
    /* Pointer to Image-1 FW metadata table. */
    cy_stc_sys_fw_metadata_t *gl_img1FwMetadata = (cy_stc_sys_fw_metadata_t *)(CY_APP_SYS_IMG1_FW_METADATA_ADDR);
    #if (!CY_APP_DUALAPP_DISABLE)
    /* Pointer to Image-2 FW metadata table. */
    cy_stc_sys_fw_metadata_t *gl_img2FwMetadata = (cy_stc_sys_fw_metadata_t *)(CY_APP_SYS_IMG2_FW_METADATA_ADDR);
    #endif /* (!CY_APP_DUALAPP_DISABLE) */
  3. Invoke the APIs to perform firmware validation and load a valid image
    /* Validate Firmware Image-1. */
    appStatus = Cy_App_Boot_ValidateFw(gl_img1FwMetadata);
    /* Load a valid firmware image. */
    Cy_App_Boot_Start();
  4. Refer the bootloader section for detailed information about usages and functionality of all other APIs

flash

  1. Include cy_app_flash.h to get access to all flash specific functions and other declarations
    #include "cy_app_flash.h"
  2. Call the following API from main to set the flash access limit
    /* Set the legal flash access range. */
    /* Set flash access limit. */
    Cy_App_Flash_SetAccessLimits ((CY_APP_IMG1_LAST_FLASH_ROW_NUM+1),
    CY_APP_IMG2_LAST_FLASH_ROW_NUM,
    CY_APP_SYS_IMG1_METADATA_ROW_NUM,
    CY_APP_BOOT_LOADER_LAST_ROW);
  3. Call the enter mode API for enabling flash write operation
    /* Flash enable write operation. */
    Cy_App_Flash_EnterMode(true, CY_APP_FLASH_IF_USB_HID, false);
  4. Invoke the following API to write data into the flash memory
    uint8_t flashWriteData[256] = {0x00};
    /* Flash enable write operation. */
    Cy_App_Flash_RowWrite((CY_APP_IMG1_LAST_FLASH_ROW_NUM+1), flashWriteData, NULL);
  5. Refer the flash section for detailed information about usages and functionality of all other APIs

instrumentation

  1. Include cy_app_instrumentation.h to get access to all instrumentation functions and other declarations in this library
    #include "cy_app_instrumentation.h"
  2. Invoke the instrumentation init and start from main function
    /* Initialize instrumentation task parameters. */
    Cy_App_Instrumentation_Init(&gl_TimerContext);
  3. Invoke the instrumentation task from main loop for handling the system faults
    /* Instrumentation task. */
    Cy_App_Instrumentation_Task();
  4. Refer the instrumentation section for detailed information about usages and functionality of all other APIs
