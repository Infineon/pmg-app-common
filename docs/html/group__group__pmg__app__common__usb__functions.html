Function requests for flashing mode operation.

The function enables/disables the billboard interface in USB flashing mode. This should be invoked only if the solution requires to start the flashing mode. This request shall prevent the billboard expiry on enable and restart the billboard expiry on disable as required.

Parameters
ptrAltModeContextPointer to the AltMode context
enableEnable/disable control. true = Enable, false = Disable.
Returns
Status of the call

◆ Cy_App_Usb_BbIsIdle()

bool Cy_App_Usb_BbIsIdle ( cy_stc_pdaltmode_context_t *  ptrAltModeContext)

Function returns whether the billboard module is idle or not.

This function indicates whether there are any pending tasks or not. This function can be invoked before device enters Sleep mode to check if it is allowed. But idle condition does not allow Deep Sleep entry. For this, the application is expected to use the Cy_App_Usb_Sleep() function.

Parameters
ptrAltModeContextPointer to the AltMode context
Returns
true = billboard module is idle; false = billboard module is busy.
