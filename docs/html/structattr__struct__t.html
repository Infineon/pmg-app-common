attr_struct_t Struct Reference

Description

Internal DMC structure which holds the parameters needed for the next FW update to be initiated/the one in progress.

This structure is updated with the information related to the next image to be updated when DMC is in CY_APP_DMC_STATE_FW_UPDATE_ANALYSIS state. Following that until DMC enters CY_APP_DMC_STATE_FW_UPDATE_ANALYSIS state again, this structure holds the information related to the image being updated currently.
