cy_stc_app_dmc_update_opern_t Struct Reference
DMC » data structures

Description

Structure holding function pointers of various device modules.

This will be filled with comp_id as index - same order as devices are in CDTT.

Warning
Below listed function pointers should not be left NULL. when new device module is being added else the code will brick or may function in an unpredicted manner. init_dev_param flash_row dev_update_logic skip_jump_to_alt_request
