cy_stc_dmc_app_cbk_t Struct Reference
DMC » Data Structures

Description

Structure to hold the application interface.

The application is expected to fill the structure with pointers to functions that use the on board circuitry to accomplish tasks.

Warning
The application must check that the callback pointer passed by the stack is not NULL.
