dmc_fwct_img_info_t Struct Reference

Description

Structure which holds FWCT parameters arrived from host.

This is analyzed in CY_APP_DMC_STATE_FWCT_ANALYSIS state (one time) whenever the FWCT data is sent by the host tool to derive the parsed FWCT structure, dmc_fwct_parsed_t.
