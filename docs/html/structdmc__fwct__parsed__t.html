dmc_fwct_parsed_t Struct Reference

Description

Internal DMC structure which holds parsed/analyzed FWCT parameters based on the FWCT image structure (dmc_fwct_img_info_t) copy present in RAM (arrived from host).

FWCT image structure (dmc_fwct_img_info_t) copy present in RAM (arrived from host) is analyzed in CY_APP_DMC_STATE_FWCT_ANALYSIS state (one time) whenever the FWCT data is sent by the host tool to determine if update is needed or not based on FW version and validity information of the device images. This structure is used every time the DMC enters CY_APP_DMC_STATE_FW_UPDATE_ANALYSIS state to find the next image to be updated.
