dmc_fwct_seg_info_t Struct Reference

Description

Structure which holds FWCT parameters arrived from host.

THis holds the image segment related information This will be present in DMC FWCT RAM buffer but unused by the FW. Hash computation during authentication happens on the entire FWCT contents including this segment info.
